## Phase 1 — Interactive Category Menu (Prioritas sekarang)

Ini perubahan terbesar dan paling impactful. Saat ini RAVEN langsung run analisis, tapi idealnya user disambut dengan menu pilihan dulu.

**Implementasi yang direkomendasikan — 2 mode:**

**Mode A: Shell `select` menu (paling simpel, native bash)**
```bash
# Di awal main(), sebelum dispatch ke Python
show_category_menu() {
    echo ""
    echo "  Pilih kategori soal CTF:"
    echo ""
    PS3="  > Masukkan nomor: "
    select choice in \
        "Steganografi (PNG/JPG/audio/LSB)" \
        "Forensik Digital (Disk/Memory/PCAP)" \
        "Kriptografi (RSA/XOR/Cipher klasik)" \
        "Web & Log Analysis (Apache/Nginx/HTTP)" \
        "Reversing (ELF/PE/bytecode)" \
        "Hash Cracking (John/Hashcat)" \
        "Encoding & Misc (B64/Hex/Brainfuck)" \
        "AI Solver (kirim ke Claude API)" \
        "Auto-detect (biarkan RAVEN memilih)" \
        "Keluar"
    do
        # set args sesuai pilihan → dispatch ke Python
    done
}
```

**Mode B: `whiptail` / `dialog` TUI** — lebih cantik, checkbox, bisa multi-select.

**Mode C: `fzf` fuzzy menu** — ketik sebagian nama kategori, langsung filter.

Rekomendasi: mulai dari Mode A dulu (zero dependency), lalu upgrade ke `fzf` jika tersedia.

## Phase 2 — Smart Auto-Detect dengan Confidence Score

Tambahkan ke Python engine: setelah scan magic byte + entropy + strings, **suggest kategori** ke user sebelum run:

```
[RAVEN] Deteksi awal:
  • PNG image terdeteksi → Steganografi (confidence: 92%)
  • LSB ratio mencurigakan (0.499) → kemungkinan hidden data
  
  Lanjutkan dengan mode Steganografi? [Y/n] atau pilih lain [1-9]
```

---

## Phase 3 — Modul Baru yang Perlu Ditambahkan

**Yang paling kritis untuk CTF:**

**Reversing module** — saat ini RAVEN tidak punya ini sama sekali:
- `strings` + `objdump` + `readelf` sudah ada di Linux
- Integrasi Ghidra headless untuk decompile otomatis
- Deteksi packing (UPX, custom) + auto-unpack

**AI Solver module** — kirim soal ke Claude API:
- Paste ciphertext → Claude analisis dan jawab
- Berguna untuk encoding eksotik (Brainfuck, Ook!, Malbolge, dll.)

**CTF Session Manager** — save/load progress:
- Simpan flag yang sudah ditemukan per challenge
- Export laporan HTML/PDF untuk writeup

---

## Phase 4 — Quality of Life

Yang paling sering bikin frustasi saat CTF: tidak tahu tool sedang apa. Tambahkan:
- **Progress bar** per tool (pakai `pv` atau spinner bash sederhana)
- **Timeout per tool** yang bisa di-set (`--timeout 30`)
- **JSON output** agar bisa di-pipe ke tool lain
- **Plugin system** — drop file `myplugin.sh` ke folder `~/.raven/plugins/`, otomatis terdeteksi
